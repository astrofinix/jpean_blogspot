Diving into Data Science: My experiences so far...
The Big Hook: Why?

Choosing Data Science has been a curveball of a path for me to choose. Medicine was my first choice of career when I was at Senior High School, and I have barely known the field of Computer Science. One of the reasons I chose medicine was because of its wide array of applications in not just saving lives but the quality of life it improves. From highly-technical surgeries, to mountains of clinical trial researchers, I was fascinated by its world applications that can make a difference to the world.

But it all changed when I asked one of my classmates what their course they will take in college and they said Computer Science. I was at first clueless as to what it was at that time, I barely recognized that there was such a field, but exploring more on it I have grown to pick it as a second choice in college. But the most important factor to pursue not only Computer Science but specifically Data Science is because it's even wider application, not just for medicine but to almost all aspects of human life, from governance, disease detection, automated driving, and to almost anything that requires processes and problems that needs answers. In simple words, Data Science is extracting valuable insights from a pool of data, in this case, data can be sourced from everything, and it has been a driving agent of innovation through the years.
The Fundamentals

Informally, Data Science suits you if you complete its ‘trifecta', the three main components that if you possess, this field is for you:

1.1 Python to Rule Them All

Data Science can be done in R, Java and even C++, but nothing beats the power and versatility of Python as the main programming languages of this field. Sometimes, you will need to learn SQL as well, but all in all, knowing and mastering Python will be a big advantage in Data Science. Not only familiarizing yourself with advanced functions python can do such as errors handling, decorators, and multithreading, but also utilizing specialized libraries for data science such as numpy, pandas, and plotly will be your best allies in harnessing its power.

For total beginners, you could always start from beginner tutorials for python to get you started and try to build a project of your own so you can further learn more skills as you encounter challenges; that is how I learned in my way. Remember, go at your own pace.

1.2 Math ain't just a number, it's applied!

Now, we've all been there when we question the applications of math in our real lives. Now that will be the reality in data science, especially these topics that will be vital to know and learn:

  • Basic Math
  • Calculus
  • Probability
  • Statistics
  • Linear Algebra
  • Linear Regression
  • Neural Networks


I recommend that book “Essential Math for Data Science: Take Control of Your Data with Fundamental Linear Algebra, Probability, and Statistics” by Thomas Nield to get you what essential math that you would be using in data science. If books aren't for you, there are always resources on youtube you can use as your tutorial. You can use the table contents of this book so you can outline the topics you need to learn.

1.3 An Observant Eye for the World

It is truth to be told that problems reside in the world, in the gaze that seeks it. This phrase does not call to turn a blinds eye for them but to gauge them to seek solutions. We are surrounded by it, for data science having a good domain knowledge for these problems can be key in spearheading the problem.

Being able to see problems in a programmatic view can entail unlimited ways of applying data science. Do you wanna know if your students are enjoying the subject? Take a survey, and you can analyze their insights. Want to detect tumors in x-rays, and possibly even better than most doctors? Create an artificial intelligence that can learn through thousands of data sets. That's data science. A person's ability to recognize problems is the first step in finding life-changing inventions no matter how subtle or innovative. It only takes to believe that you can be the solution to make it happen.
Developing Experience

Now, the developer tool specifically the code editor I am using is VS Code (Visual Studio Code). This software is no longer just for editing source codes but has grown to be a robust and powerful tool to use not only for data science but for other development fields such as Web Development among others. This program is further empowered by the number of extensions you can use in your programming. Here are the list of extensions I use:

Code Runner - Automatically run code without manually doing it on terminal
Live Share - Online collaboration while coding
Jupyter - Jupyter Notebook support for VS Code
Path Intellisense - Detect the directories you can use in your code

Also, swaying away from the old-fashioned user interface of Code editors is, in my opinion, painful in the eyes. Here are some customization I use in VS Code to further make its user interface aesthetic:

Personal Recommendations
Jetbrains Mono as the Editor's Font - legible and stunning at the same time, no more boring font.
Material Icon theme - beautiful icons that labels your file types
Atom One Dark Pro Theme - For rich dark theme for VS Code
AutoPep8 - Automatically format your code in PEP8 (formatting convention in Python)
Rainbow CSV - Label CSV context
Getting Help Will Be Your Best Ally

Now, there are challenges that arise coming as unfamiliar or unsolvable, but don't worry, the wisdom from thousands of developers from the past is kept in some corners of the internet, namely on forums and sites. Here's are the two places I go to whenever I have some bug or a problem I need to fix, (hehehe tambayan sa mg Machine Problems and Exercises)


Search Tip: for the first word of your search query add the site you want to source your answer and then problem you want to solve


Example:
stackoverflow css center an image
Reddit pandas save dataframe on csv


StackOverflow - This place is filled with lots of questions and surprisingly a million users that come from 14 years ago. But be warned, post a question if and only if that question hasn't been asked in the site in its history. And I am not exaggerating, your post will receive negative votes and possibly disable you from ever posting a question in the site. This mechanism mitigates the duplicated questions. Just make sure that the problem you are asking is really unique and hasn't been asked before.


Reddit - Now, for this site I recommend r/learnprogramming, and depending on your question you can visit r/learncss, r/learnpython, or r/learnjava, these communities are quite friendly to beginners and less restrictive in the questions you ask. I can't stress enough how these communities have helped me in my programming experience.But do make sure the formatting rules and regulation from each community, this is not limited to the format of your questions, and the tags you use.
TL; DR (Too Long; Didn't Read.)

If you have a good eye for problems around you, you enjoy applied math, and you have extensive programming skills, data science may be for you. It is a field in creating insights from everything around you (data). These insights can turn into solutions.


My personal recommendation is to start small, get started with tutorials and for your developing experience, VS Code can be your best ally especially if you know how to customize it to your liking.


Lastly, the internet has become a life-changing tool for everyone, utilize it. Asking for help will be one of the essential skills in learning a lot of things, not just data science.
